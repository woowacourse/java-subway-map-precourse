# 지하철 노선도 미션
## 기능 구현 목록
### 1. 메뉴 출력 기능 구현
#### 메인 메뉴
- 예외 1) 1-4, Q 이외의 입력을 받는 경우
- 예외 2) 아무 입력 없는 경우 
  
#### 역 관리
- 1 - 역 등록
    - 예외 1) 중복된 역 이름을 입력 받는 경우
    - 예외 2) 1글자 이하를 입력 받는 경우
- 2 - 역 삭제
    - 예외 1) 노선에 등록된 역인 경우
    - 예외 2) 등록되지 않은 역인 경우
    - 예외 3) 아무 입력 없는 경우
- 3 - 역 조회
    - 등록된 순서대로 출력
    
예외 1) 정수 1~3, B 이외의 입력을 받는 경우 <br>
예외 2) 아무 입력 없는 경우
      
#### 노선관리
- 1 - 노선 등록
    - (1) 상행종점역을 입력 받는다.
    - (2) 하행 종점역을 입력 받는다.
    - 예외 1) 중복된 노선인 경우
    - 예외 2) 1글자 이하인 경우
 
- 2 - 노선 삭제
    - 예외 1) 등록되지 않은 노선인 경우
    - 예외 2) 아무 입력 없는 경우

- 3 - 노선 조회
    - (1) 현재 등록된 노선 출력
    
예외 1) 정수 1~3, B 이외의 입력을 받는 경우 <br>
예외 2) 아무 입력 없는 경우

#### 구간 관리
- 1 - 구간 등록
    - 노선 입력
        - 예외 1) 존재하지 않는 노선인 경우
        - 예외 2) 아무 입력 없는 경우
    
    - 역이름 입력
        - 예외 1) 존재하지 않는 역이름인 경우
        - 예외 2) 아무 입력 없는 경우
        - 예외 3) 이미 등록된 역이름인 경우
    
    - 순서 입력
        - 예외 1) 1 ~ (노선의 길이-1) 사이의 정수가 아닌 경우
        - 예외 2) 아무 입력 없는 경우
    
- 2 - 구간 삭제
    - 삭제할 구간 노선 입력
        - 예외 1) 존재하지 않는 노선인 경우
        - 예외 2) 아무 입력 없는 경우
    - 삭제할 구간 역 입력
        - 예외 1) 입력 받은 노선에 해당 역이 없는 경우
        - 예외 2) 상행 종점역 혹은 하행 종점역을 입력 받은 경우
        - 예외 3) 노선에 포함된 역이 두개 이하인 경우
        
#### 노선도 출력
- 노선 정보 출력
- "---" 출력
- 노선에 존재하는 역 차례대로 출력
