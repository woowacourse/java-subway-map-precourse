기능 구현 사항

- 사용자의 input
    1. 사용자가 원하는 기능을 입력한다.
- 사용자의 output
    1. 메인 메뉴를 출력한다.
    2. 역 관리 메뉴를 출력한다
    3. 노선 관리 메뉴를 출력한다.
    4. 구간 관리 메뉴를 출력한다.

- 지하철 역 관리
    1. 역 관리 화면 출력과 기능 연결
    2. 역을 등록 또는 삭제 할 수 있다.

- 지하철 노선 관리
    1. 노선 관리 화면 출력과 기능 연결
    2. 노선을 등록 또는 삭제 할 수 있다.

- 지하철 구간 관리
    1. 구간 관리 화면 출력과 기능 연결
    2. 구간을 등록 또는 삭제 할 수 있다.
